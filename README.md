# HY Canvas Minimap

A minimap for [Obsidian](https://obsidian.md) Canvas. Shows an overview of the
whole canvas in a corner, with a viewport box you can click or drag to move
around. Double-click to zoom to the nearest node.

## Settings

- Show/hide the minimap
- Corner: bottom-left or bottom-right
- Size (as a percent of the canvas view)

> Obsidian's Canvas has no public API, so this plugin relies on internal canvas
> objects — a future Obsidian update could break it.
